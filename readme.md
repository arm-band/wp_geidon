# Sukunabikona

## Abstract

`wp-env` を使用した最小限の開発環境

## Using

## first

1. `npm i -D` or `yarn`
2. `wp-env start`
    - stop: `wp-env stop` or `npm run wp:stop`
    - `ctrl` + `c`でタスクを終了させた場合も`wp-env stop` or `npm run wp:stop`でDockerコンテナを止めること
3. `npm start`

## after second

1. `wp-env start`
    - stop: `wp-env stop` or `npm run wp:stop`
    - `ctrl` + `c`でタスクを終了させた場合も`wp-env stop` or `npm run wp:stop`でDockerコンテナを止めること
2. `npm restart`
